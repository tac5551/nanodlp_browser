NanoDLP Browser

# DEMO
 
"hoge"の魅力が直感的に伝えわるデモ動画や図解を載せる
 
# Features

By using UPnP, 3D printers with NanoDLP installed on the network can be automatically detected and centrally managed.
Multiple printers can also be monitored at once.

UPnPを利用して、ネットワーク上に接続されているNanoDLPが導入された3Dプリンターを自動的に検出し、一元管理が可能です。
複数のプリンタも一度に監視することができます。
 
# Requirement

* Windows8,10,11
* .net Framework4.5.1 Runtime
 
# Installation

Just run "NanoDLP_Browser.exe".
No installer is available for the current version.

NanoDLP_Browser.exeを実行してください。
現在のバージョンではインストーラーを用意していません。
 
# Usage

・Double-click on the printer in the list to display the NanoDLP screen in your browser.
・When the window is closed, it is stored in the task tray.
・When you close the window, it will be stored in the task tray. - You will be notified when printing is finished or a new printer is connected.
![](https://tac-lab.tech/wp-content/uploads/2022/02/POPUP.png)


・一覧に表示されたプリンタをダブルクリックすると、ブラウザでNanoDLPの画面が表示されます
・ウィンドウを閉じると、タスクトレイに格納されます。
・印刷が終了したり、新しいプリンタが接続された場合に通知されます
